# Drinking Leaderboard

Willkommen beim Drinking Leaderboard Projekt!

## Projektstruktur

```
Drinking_Leaderboard/
├── frontend/          # Next.js Frontend-Anwendung
├── backend/           # Backend API und Server-Logik
└── database/          # Datenbankschemas, Migrationen und Seeds
```

## Frontend

Das Frontend basiert auf Next.js mit React und TailwindCSS.

**Entwicklungsserver starten:**

```bash
cd frontend
npm install
npm run dev
```

Der Frontend-Server läuft dann auf [http://localhost:3000](http://localhost:3000).
